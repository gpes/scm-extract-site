scm-extract para Java




O que é ?

O scm-extract é uma ferramenta desenvolvida em Java que visa a automatização do processo de checkout de projetos de software, bem como de suas versões, a partir de repositórios online de códigos, como por exemplo o Github. Esta ferramenta surgiu a partir da necessidade de se obter várias versões de um projeto de software de forma mais rápida, para que assim pudessem ser realizadas pesquisas sobre tais versões. O scm-extract atualmente conta com a integração com dois scm (gerência de configuração de softare, do inglês, software configuration management), Git e Svn. Em suas próximas versões contará com a integração ao Mercurial.

Getting Started

Clone o repositorio 
git clone https://github.com/EndenhariaDeSoftware/scm-extract

Usando maven


Execute 
mvn clean install

Adicione como dependencia local 
mvn install:install-file -Dfile=./target/scm-extract-1.0-SNAPSHOT.jar -DgroupId=br.edu.ifpb.scm -DartifactId=scm-extract -Dversion=1.0 -Dpackaging=jar

Abra um projeto na IDE e adicione a dependência do scm-extract no pom.xml 
<dependency>
 	<groupId>br.edu.ifpb.scm.api</groupId>
 	<artifactId>scm-extract</artifactId>
 	<version>1.0</version>
</dependency>

Exemplos de uso

  • Clonando um repositório remoto
  • Recuperando um repositório local
  • Exibindo as informações de commits
  • Exibindo quais arquivos foram modificados entre as versões
  • Exibindo as modificações entre as versões dos arquivos
